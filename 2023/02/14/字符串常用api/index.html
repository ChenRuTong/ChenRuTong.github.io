字符串常用api

cc_chen lv1
  2023-02-14 10:42:52 2023-02-14 10:42  
  • javascript 

1.slice()

不会改变原字符串，返回被截取后的字符串

2.split()

不会改变元字符串，返回分割成的数组

split(delimiter [,limit])

delimiter：分割处的字符串或正则表达式。
limit: 指定已返回数组的最大长度。

3.indexOf()

indexOf(value [,from])
在指定字符串中寻找一个字符或子串。只查找第一个，找不到返回 -1。

4.lastIndexOf()

lastIndexOf(value [,from])
在指定字符串中寻找一个字符或子串。只查找第一个，找不到返回 -1。

5.concat()

concat(value)

把value拼接到str上

不会改变原字符串，返回拼接后的新字符串

6.charAt()

charAt(index) 和 str[index]

获取指定位置字符。

7.trim()

去掉字符串前后空格。

7.charCodeAt()

charCodeAt(index)

指定位置的字符转ASCII码 （返回的是一个整数，值在 0~65535 之间）

如果 index 为负数或大于等于字符串的长度，则 charCodeAt() 将返回 NaN。

8.fromCharCode()

fromCharCode(c1, c2, …)

跟 str.charCodeAt() 相反的操作，把 ASCII码 转成字符串。

9.toUpperCase()

转成大写 (不会改变原字符串，返回新的字符串)

10.toLowerCase()

转成小写 (不会改变原字符串，返回新的字符串)

11.includes()

includes(searchvalue[, start])

用于判断字符串是否包含指定的子字符串。
如果找到匹配的字符串则返回 true，否则返回 false。

searchvalue 必需，要查找的字符串。
start 可选，设置从那个位置开始查找，默认为 0。

12.replace()

str.replace(regexp|string, newString|function)

替换（不会改变原字符串，返回替换后的字符串）

如果第2个参数是函数，该函数的返回值将作为替换文本
